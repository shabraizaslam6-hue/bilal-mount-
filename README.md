# My Website

A modern, responsive website template built with HTML and CSS.

## Features

- ✨ Clean and modern design
- 📱 Fully responsive (mobile, tablet, desktop)
- 🎨 Beautiful gradient colors
- ⚡ Smooth animations and transitions
- 📧 Contact form section
- 🚀 Easy to customize

## Quick Start

1. Clone the repository
2. Open `index.html` in your browser
3. Customize the content with your own information

## Sections

- **Home** - Hero section with call-to-action button
- **About** - About you or your business
- **Services** - Display your services or features
- **Contact** - Contact form for visitors

## Customization

- Edit `index.html` to change content
- Edit `style.css` to change colors and styling
- Replace placeholder text with your own content
